# ngx-comparison-slider
Image comparison slider component for Angular

# How to use it
```HTML
<app-ngx-comparison-slider
  [preImageUrl]="'http://lorempixel.com/600/400/sports/1'"
  [postImageUrl]="'http://lorempixel.com/600/400/sports/2'"
  [sliderPosPct]="80"
>
</app-ngx-comparison-slider>
```

Optional parameters
 - sliderPosPct
